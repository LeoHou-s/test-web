=== CSS 的三種使用方式 ===
  • 行內樣式 (Inline Styling)

    直接在 p 後面加上 style="..."

  • 內嵌樣式 (Embedded Styling)

    在 head 裡加入 style...

  • 外部樣式表 (External Stylesheet)

    在外部新增一個新的 .css

=== CSS 的顏色 ===
  1. 內建顏色

    color: red

  2. RGB 自訂顏色

    color: rgb(R[數字], G[數字], B[數字])

  3. 16進制顏色

    color: #六位數[0123456789abcdef]

  4. HSL

    H 色調

    S 飽和度

    L 亮度

    color: rgb(R[數字], G[數字], B[數字])

=== CSS 字體 ===
  • 字體樣式(字體找不到會用第下一個)

    font-family: 字體1,字體2,字體3

  • 字體大小(pixel, em百分比)

    font-size: px

  • 字體粗細(normal:一般, bold:粗體, 數字)

    font-weight: [normal, blod, 100]

  • 字體效果(italic斜體)

    font-style: italic

邊框 border

基本邊框

  • border-style: 類型[solid, dot,...];

  • border-width: 大小(px);

  • border-color: 顏色;

  • border-radius: 大小(px);

邊框縮寫語法

border: [類型] [大小] [顏色]

border: solid 5px red

四邊邊框

  • border-top: 1px solid red;

  • border-bottom: 5px solid green;

  • border-left: 3px dashed blue;

  • border-right: 2px dashed #000000;
