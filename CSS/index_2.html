CSS 的三種使用方式

    在外部新增一個新的 .css

  2. RGB 自訂顏色

    H 色調

height 與 width

用法

width height

  • width (絕對/相對寬度)： 強制寬度。例如 width: 25% 代表永遠佔 1/4

  • min-width (最小寬度)： 這是「底線」。不管父容器多小，盒子都不能縮得比這個更小。當內容（如你設定的 80px 字體）太多時，min-width 能防止內容溢出崩壞。

  • max-width (最大寬度)： 這是「天花板」。不管父容器多大，盒子都不能長得比這個更大。這常用於防止圖片或段落在大螢幕上顯得過寬。

vh

100vh (Viewport Height)： 代表「螢幕高度的 100%」

box-sizing: border-box

box-sizing: border-box

  • 預設情況 (content-box)： 如果你設定寬度 25% 再加 3px 邊框，實際寬度會變成 25% + 6px，這會導致四個盒子加起來超過 100% 而掉下

  • 使用 border-box： 寬度會包含邊框和內距。25% 就是整體的總寬，這讓計算排列變得超級簡單。
